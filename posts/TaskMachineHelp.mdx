# 帮助文档
## 任务机器的使用场景
任务机器中的任务主要指**需要花大量时间的，需要重复的，需要花费精力的**任务。比如学习，运动。任务机器不适合其他任务：比如出门拿个快递。
## 开始使用
1. 将任务机器App比作游戏里的NPC，它的作用就是根据当前你所处的时间地点为你提供一个合适的任务，减少不必要的精神内耗，你无需自己做安排，让你更加专注于当下。不过任务机器并不能无中生有，你得先添加任务……
2. 添加任务，为每一个任务设置他们的条件以及重要程度，重复此步骤即可。
3. 告诉任务机器你在哪里，他就会给你提供一个合适的任务，调节预计时间，点击开始即可。
4. 长按"X"提前结束。
## 最合适的任务
1. 这是这个软件的核心，当你告诉任务机器你在哪里之后，任务机器App会通过当前时间，以及你选择的地点，筛选出**符合当前条件的**任务，然后在这些筛选出的任务中，根据每一个任务上一次做到现在的**时间跨度**及其**重要程度**，计算出一个最合适的任务。当你点击开始后，任务机器会将该任务的**时间跨度**刷新为0，随着时间的推移，时间跨度又会不断增加，以此作为下一次计算的依据。
2. 所以，任务机器App所计算出的任务一定是一个**有一段时间没有做了的，重要程度相对较高的，符合当前时间地点的任务。**
3. 不过，无论机器如何计算，当前最该做什么，只有使用者自己最清楚，因此，你可以在推荐任务视图的右上角切换到其他符合当前条件的任务。
## 快捷指令
1. 在设置中打开该选项，并输入一个快捷指令的名称，然后在快捷指令App中创建一个相同名字的快捷指令。在开始一个任务时，就会自动运行该快捷指令。
2. 在输入快捷指令的信息中，包含了这样一个词典：你可以利用该信息结合快捷指令来实现许多个性化的功能。

|键|值|
| ---- | ---- |
|标题|当前任务的标题|
|类别|当前任务的父任务标题|
|重要程度|当前任务的重要程度|
|时长|当前任务所预计的时间|

# User Guide

## Usage Scenarios of Task Machine

The tasks in Task Machine mainly refer to tasks that require a lot of time, repetition, and energy. For example, studying and exercising. Task Machine is not suitable for other tasks, such as going out to pick up a delivery.

## Getting Started

Think of Task Machine App as an NPC in a game. Its role is to provide you with a suitable task based on your current time and location, reducing unnecessary mental energy consumption. You don't need to arrange things yourself, allowing you to focus more on the present moment. However, Task Machine cannot create something out of nothing. You need to add tasks first...
Add tasks and set their conditions and importance level for each task. Repeat this step for each task.
Tell Task Machine where you are, and it will provide you with a suitable task. Adjust the estimated time and click start.
Long press "X" to end the task early.
## Most Suitable Task

This is the core feature of this app. After you tell Task Machine where you are, it will filter out the tasks that meet the current conditions based on the current time and the location you selected. Then, among these filtered tasks, it will calculate the most suitable task based on the time span and importance level of each task from the last time it was done until now. After you click start, Task Machine will refresh the time span of the task to 0. As time passes, the time span will continue to increase, serving as the basis for the next calculation.
Therefore, the task calculated by Task Machine App must be a task that has not been done for a period of time, has a relatively high importance level, and meets the current time and location.
However, no matter how the machine calculates, only the user knows best what they should do at the moment. Therefore, you can switch to other tasks that meet the current conditions in the upper right corner of the recommended task view.
## ShortCuts

Turn on this option in the settings and enter a name for the quick command. Then create a quick command with the same name in the Quick Command App. The quick command will automatically run when you start a task.
The information input for the quick command includes a dictionary: you can use this information in combination with the shortcuts to achieve many personalized functions.
 |Key |	Value |
| ---- | ---- |
 |Title	 |The title of the current task |
 |Category	 |The parent task title of the current task |
 |Importance Level	 |The importance level of the current task |
 |Duration	 |The estimated time for the current task |
